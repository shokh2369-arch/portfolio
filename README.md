# Shohjahon Abdullayev — Portfolio

Personal portfolio site for a Backend / Full-Stack Developer (Go & Node.js).

## Stack

- React 19 + Vite 8
- Tailwind CSS 4
- Framer Motion

## Run locally

```bash
npm install
npm run dev
```

Open [http://localhost:5173](http://localhost:5173).

## Build

```bash
npm run build
npm run preview
```

## Customize

- **Links & content:** `src/data/content.js`
- **CV:** replace `public/cv.pdf` with your real resume
- **GitHub username:** update URLs in `content.js` if needed
